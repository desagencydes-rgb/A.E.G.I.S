# AEGIS - ChatDev Bridge

This bridge enables AEGIS agents to run as custom nodes within ChatDev workflows.

## Structure

- `aegis_node.py` - Main custom node implementation
- `config.py` - Configuration settings
- `memory_bridge.py` - Memory synchronization
- `permission_enforcer.py` - Permission system integration

## Usage

See [`../workflows/`](../workflows/) for example YAML configurations using AEGIS agents.
